Подготовить техническое задание (ТЗ) в соответствии с ГОСТ 34, которое станет приложением к договору с подрядчиком по созданию информационной системы. Вам необходимо как можно более полно описать задачу автоматизации — от этого зависит, насколько хорошо система сможет удовлетворить бизнес-потребности вашей компании и поможет лично вам быстрее и лучше выполнять свою работу. Хорошее ТЗ требует тщательного обдумывания и детальной проработки, поэтому вам следует декомпозировать задачи по его написанию и распределить между членами команды по справедливости.
